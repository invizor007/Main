Формат HIG файла

Heroes imitation game- файл сохраненной игры, состоит за карты и дополнения к ней- например текущего дня, ресурсов, видимости Сначала сохраняется сама карта(формата him) на момент игры, потом сохраняется следующее:
//Основное 
Количество_Активных_Игроков
к.а.и раз НомерАктивногоИгрока
ТекущийНомерАктивногоИгрока День Неделя Месяц
//Видимость
Для всех игроков и всех координат хранится значение видимо 1-невидимо 0 через enter //Наличие миссий шпионов, для каждого игрока по всем активным объектам на каждой строке через пробел
Герой-3 миссии
Шпион-1 миссия
Строитель-3 миссии
//У торговца созраняемых миссий нет
Рабочий-2 миссии
У посла-1 миссия
У Зеркала 1 миссия
У инженера 2 миссии
//Ресурсы 
Золото Дерево Камень Железо Мифрил Нефть Наука
Ресурсы города(для каждого города каждого игрока) 
Рабочие Еда
Текущие показатели действующих объектов Показанные места артефактов обелисков через enter (Для 8 игроков, 10 номеров обелисков, 32 потенциальных мест)
